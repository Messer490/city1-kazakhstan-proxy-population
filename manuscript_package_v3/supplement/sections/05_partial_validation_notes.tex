\section{S5. Notes on Partial Phase 6 Validation}

Phase 6 was completed only partially, and the manuscript carries that status explicitly.

First, the held-out LOCO-like evidence was generated in a bounded local configuration with 5 ensemble members and 60 trees per member. This means the retained interval-coverage and error-alignment results are scientifically useful but should not be overstated as if they came from the full 20-member runtime package.

Second, true district interval coverage remains blocked in the light package because frozen district polygon/cell assignment artifacts are unavailable. The manuscript therefore uses district evidence only as partial after-aggregation support.

Third, external disagreement alignment against WorldPop and GHS-POP is mixed or negative in the frozen package. This does not invalidate the uncertainty layer, but it prevents any claim that uncertainty systematically rises wherever City1 disagrees with external products.

Taken together, these notes justify the main-paper framing of City1 v3 as an uncertainty-aware screening and interpretation layer rather than a fully validated census-uncertainty model.
