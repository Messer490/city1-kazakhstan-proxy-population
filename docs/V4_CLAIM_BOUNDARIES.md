# City1 v4 Claim Boundaries

## Allowed claims

- City1 produces a calibrated proxy population surface.
- V3 adds uncertainty-aware interpretation.
- P10/P50/P90 summarize ensemble spread inside the proxy framework.
- `confidence_score` is an interpretation-confidence score.
- Hotspot classes are screening and triage categories.
- External products are structural comparators.
- V4 improves interpretability, usability, evidence linking, and claim-boundary discipline.
- V4 is evidence-grounded and does not change the underlying model outputs.

## Forbidden claims

- City1 reconstructs true cell-level census counts.
- P10/P50/P90 are true census uncertainty intervals.
- `confidence_score` is a probability of correctness.
- WorldPop or GHS-POP are ground truth.
- `hotspot_priority_class` is verified population-hotspot truth.
- V4 improves population prediction accuracy.
- The LLM creates or changes population estimates.
- City1 can be used for definitive administrative accounting.
- City1 supports fully automated policy decisions.

## Practical wording rules

- Prefer: `supports`, `helps interpret`, `can inform`, `screens`, `summarizes`.
- Avoid: `proves`, `verifies`, `accurately reconstructs`, `truth`, `certainty`, `probability`.
- If a statement could be read as truth-recovery, rewrite it as proxy interpretation.
