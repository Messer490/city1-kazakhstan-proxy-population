# Submission Metadata Checklist

These items are intentionally kept separate from the frozen scientific content because they depend on author, venue, or project-specific decisions rather than on the `v2` evidence stack.

## Required before submission

- author names
- affiliations
- corresponding author marker
- keywords
- funding statement, if applicable
- acknowledgements, if applicable
- conflict of interest statement, if required by the venue
- target journal template migration

## Practical rule

Do not edit scientific claims, figures, or tables while filling these fields. This checklist is for submission metadata only.
