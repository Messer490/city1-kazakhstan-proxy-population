\section*{Data and Code Availability}

The City1 v2 codebase and manuscript package are available in the public repository \url{https://github.com/Messer490/city1-kazakhstan-proxy-population}. This manuscript is tied to commit \texttt{92cc78d7880af1388fcbf3a5d7aa8db57c5c4512}. A long-term archival DOI has not yet been assigned and should be inserted here after repository deposit: \texttt{[archival DOI to be added]}.

The main paper and supplement are generated from the canonical v2 paper-facing package in \texttt{reports/paper\_v2\_baseline/}, using the frozen workflow scripts \texttt{scripts/train\_model\_v2.py}, \texttt{scripts/predict\_city\_v2.py}, \texttt{scripts/run\_district\_benchmark\_v2.py}, \texttt{scripts/run\_external\_benchmark\_v2.py}, \texttt{scripts/run\_ablation\_study\_v2.py}, \texttt{scripts/run\_qualitative\_validation\_v2.py}, and \texttt{scripts/build\_paper\_report\_v2.py}. The canonical official-total reference is \texttt{data/external/city\_population\_reference\_v2.csv}, and the frozen city-support registry is \texttt{data/external/city\_status\_registry\_v2.csv}. District benchmark inputs are retained in \texttt{data/external/district\_benchmark/}, while the integrated paper-facing tables and figures used for the manuscript are retained in \texttt{reports/paper\_v2\_baseline/}.

Official city totals are compiled within the v2 workflow from Bureau of National Statistics of Kazakhstan sources and fixed to the reference date 2026-02-01; Supplementary Table~S2 records the provenance route for each city. Expanded qualitative cases, the validation-evidence matrix, and technical appendix material on the city registry, feature inventory, weak-target weights, and calibration fallback behavior are provided in the supplement rather than in the main manuscript. Third-party official source files and external benchmark inputs remain subject to their original access conditions. The manuscript therefore reflects a fixed analytical baseline rather than an evolving experimental system.
